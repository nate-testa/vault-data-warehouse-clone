# DocuClaims AI - Flask UI

This is a Flask-based user interface for the DocuClaims AI application. It provides the same functionality as the original Streamlit UI but is built using Flask.

## Features

- Home page with introduction and disclaimer
- DocuClaims AI page with:
  - File upload functionality
  - Chat interface for asking questions about uploaded documents
  - Model selection
  - Theme customization
  - Chat history management

## Setup and Installation

1. Ensure you have Python 3.8+ installed on your system.

2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

3. Configure environment variables:

Create a `.env` file in the root directory with the following content:

```
API_BASE_URL=http://localhost:8000
```

Adjust the URL to point to your FastAPI backend service.

## Running the Application

Start the Flask application with:

```bash
python app.py
```

The web interface will be accessible at `http://localhost:5000`.

## Project Structure

- `app.py`: Main Flask application file
- `utils_logging.py`: Logging utility
- `templates/`: HTML templates for the UI
  - `base.html`: Base template with common layout
  - `home.html`: Home page template
  - `docuclaims.html`: DocuClaims AI interface template
- `static/`: Static assets for the UI
  - `css/`: CSS stylesheets
  - `js/`: JavaScript files
- `logs/`: Log files generated by the application

## API Integration

This UI communicates with a FastAPI backend for:

- Uploading documents
- Querying document content
- Retrieving available AI models
